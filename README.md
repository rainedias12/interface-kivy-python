# Repositório de Estudos: Interface Gráfica com Kivy e Python

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Kivy](https://img.shields.io/badge/Kivy-1F1F1F?style=for-the-badge&logo=kivy&logoColor=white)

Este repositório centraliza meus projetos práticos e códigos de estudo focados no desenvolvimento de **Interfaces Gráficas (GUI)** utilizando a linguagem Python e o framework Kivy. 

O objetivo é documentar a minha evolução na criação de componentes visuais, gerenciamento de layouts e manipulação de eventos.

 📁 Projetos Inclusos

 1.  Layout Cores Aleatórias (`Layout Cores Aleatórias.py`)
Demonstração de estrutura dinâmica onde múltiplos botões são gerados automaticamente através de laços de repetição.
* **Destaques:** Uso do componente `BoxLayout` para organização e da biblioteca nativa `random` para alternar as cores de fundo dos botões de forma dinâmica.

 2.  Botão Interativo e Eventos (`Button.py`)
Exemplo prático focado na base da programação orientada a eventos dentro do framework Kivy.
* **Destaques:** Configuração de tamanho e posicionamento proporcional de componentes na tela (`size_hint` e `pos_hint`) e vinculação de funções a ações de clique utilizando o método `bind`.

 3.  Estrutura Base de Janela (`Layout Inserir Imagem.py`)
Uma variação estrutural focada na arquitetura limpa de inicialização de uma aplicação Kivy.
* **Destaques:** Uso da classe principal `MainApp` herdando de `App` para renderizar componentes interativos e capturar logs diretamente através do terminal ao interagir com a interface.

---

 Tecnologias Utilizadas
* **Python** (Lógica, loops e manipulação de eventos)
* **Kivy** (Construção das interfaces visuais)
