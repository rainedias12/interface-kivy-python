# Interface Gráfica com Kivy e Python

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Kivy](https://img.shields.io/badge/Kivy-1F1F1F?style=for-the-badge&logo=kivy&logoColor=white)

Projeto prático desenvolvido para consolidar o aprendizado em **Interface Gráfica (GUI)** utilizando a linguagem Python e o framework Kivy. 

O foco principal do repositório é demonstrar o entendimento sobre a estruturação de layouts dinâmicos e o comportamento de componentes visuais básicos.

 🚀 Funcionalidades Demonstradas
* **Layouts Responsivos:** Utilização do componente `BoxLayout` para organização sequencial de elementos na tela.
* **Interatividade Dinâmica:** Uso da biblioteca nativa `random` para alternar cores de fundo dos botões de forma dinâmica e automatizada.
* **Laços de Repetição em GUI:** Criação estruturada de múltiplos botões através de loops iterativos.

 🛠️ Tecnologias Utilizadas
* **Python** (Lógica principal e manipulação de listas)
* **Kivy** (Framework para construção da interface visual)
